게임 피드백 품질을 점검해줘. 아래 체크리스트를 기준으로 현재 코드를 분석하고, 문제가 있는 항목은 수정 방법까지 제안해줘.

## 체크리스트

### 히트 판정
- [ ] 히트존 크기(`hitZoneSize`: 현재 60px)가 체감상 너무 좁거나 넓지 않은지
- [ ] 히트존 하이라이트(흰 원)가 실제 판정 범위와 시각적으로 일치하는지
- [ ] 조개가 히트존을 통과할 때 반응 속도가 자연스러운지 (miss 처리 타이밍)

### 입력 반응
- [ ] 탭/클릭 후 스프라이트 전환이 즉각적인지 (현재 0ms 지연)
- [ ] 400ms 후 기본 포즈 복귀 타이밍이 너무 길거나 짧지 않은지
- [ ] 터치/클릭 중복 방지 100ms 쿨다운이 과하게 입력을 막지 않는지

### 효과음
- [ ] hit/miss SFX가 시각 피드백(스프라이트 전환)과 동시에 재생되는지
- [ ] 음량 균형이 BGM 대비 적절한지
- [ ] 빠른 연속 탭 시 효과음이 끊기거나 중복되지 않는지

### 레벨 진행
- [ ] BPM 상승 시 조개 속도가 음악과 계속 동기화되는지
- [ ] 레벨업 순간 조개 스폰 공백(4비트 freeze)이 플레이어에게 자연스럽게 느껴지는지
- [ ] 레벨 표시 텍스트 팝업 타이밍이 BPM 변경과 일치하는지

### 시각 피드백
- [ ] 조개 fade-out(300ms) 속도가 자연스러운지
- [ ] 수달 float 애니메이션이 게임 중에도 방해가 되지 않는지

분석 후 우선순위가 높은 문제부터 정리해줘.
